Investigating Hotel Booking Demand¶

by Khaled Salem¶

Preliminary Wrangling¶

This dataset was downloaded from Kaggle and is available through this link https://www.kaggle.com/jessemostipak/hotel-booking-demand. It is originally from an article by Nuno Antonio, Ana Almeida, and Luis Nunes for Data in Brief, Volume 22, February 2019.

This data set contains booking information for a city hotel and a resort hotel made worldwide, and includes information such as when the booking was made, duration of stay, the number of adults, children, and/or babies, and the number of available parking spaces, among other things from 2015 to 2017.

# Defining a new function that takes on arguments to create a histogram 
def histogram(column, title, binsize, xlabel):
    # make space
    plt.figure(figsize = (16,8))
    plt.subplot(1,2,1)
    binsize = np.arange(0, df[column].max()+binsize,binsize)
    plt.hist(data=df, x = column, bins = binsize)
    plt.title('Distribution of {}'.format(title))
    plt.xlabel(xlabel)
    
    plt.subplot(1,2,2)
    plt.hist(data=df, x = column, bins = binsize)
    plt.title('Log distribution of {}'.format(title))
    plt.xscale('log')
    plt.xticks([10,30,100,300,1000])
    plt.xlabel(xlabel)

Another inference is that as lead time increases, number of bookings drops and hence most bookings are made within 0 to 100 days prior to arrival

Transformations¶

Variables that consist of quantitative data were investigated graphically, most of which appeared to be skewed to the right. In some cases, the skewness in those variables were mainly attributed to the outliers. Log transformations were performed to conform the data into normality.

Tidying operations¶

Firstly, the company column was dropped since it had many missing values making it interpretable and hence was removed. Additionally, missing values in the agent and children variables were replaced with their median whereas the missing values in country were replaced with the mode which was found to be PRT (or Portugal)
